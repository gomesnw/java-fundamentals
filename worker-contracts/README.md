# Worker Contracts (Java)

This project is a simple Java application to manage workers and hourly contracts. It calculates
the worker’s income for a given month and year, based on base salary and associated contracts.

## Concepts Demonstrated

- Object-Oriented Programming (OOP)
- Enum usage (`WorkerLevel`)
- Composition
- List manipulation
- Date handling (`LocalDate`, `YearMonth`)
