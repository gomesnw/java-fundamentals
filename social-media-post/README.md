# Social Posts

Simple social media post simulation developed to practice object-oriented programming concepts in Java.

## Description

This project models a social media post with comments, using composition to represent
the relationship between posts and comments. It also applies StringBuilder to generate
formatted output.

## Concepts practiced

- Object-Oriented Programming (OOP)
- Composition
- Encapsulation
- StringBuilder
- Date and Time API (Instant)

## Notes

This project was developed as part of my Java fundamentals studies.
